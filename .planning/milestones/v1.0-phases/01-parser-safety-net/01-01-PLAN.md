Where production code throws today, the test asserts the throw so the suite stays GREEN with zero skips; Phase 2 (BUG-03/BUG-05) deliberately inverts each named characterization to `Assert.Null` once `TryParse` + `InvariantCulture` land.
Output: Extended `GameObjectTests.cs` and `CombatLogLineTests.cs` — new tests only, no production change.
</objective>
